📦 Warehouse Production Management System

This software was built for a real warehouse client to manage all stages of inventory, production, and Amazon shipment flows. It’s one of my biggest technical achievements, and fully deployed as a secure desktop app using Electron.js.

🎥 Demo Videos

🔍 Key Features

  • ✔️ Composite Product logic (parent-child structure with accessory deduction)
  • ✔️ Real-time packaging progress with UI indicators
  • ✔️ Amazon shipment flow: Created → Prepared → Completed
  • ✔️ Partial stock receiving & reserved inventory handling
  • ✔️ Full installable app via Electron (desktop-ready)

🛠️ Technologies Used

React.js, Node.js, Express, MongoDB Atlas, Material UI, Electron.js, Webpack, Docker, GitHub

📈 Impact

Reduced warehouse packaging and tracking effort by 70%. Streamlined workflow and reduced errors significantly. Built with scalability and offline security in mind.
